# Sales Management (Demo)

This is a sample sales management web project built with basic HTML and CSS, suitable for learning purposes or user interface demonstration.

## Features

- **Product list**: Display products with prices and "Add to cart" buttons.
- **Shopping cart**: Shows selected products, quantities, total price, and "Order" button.
- **Order management**: List of placed orders, customer information, and delivery status.
- **User-friendly layout**: Modern and easy-to-use interface design.

## Usage

1. **Create Files**:
    - `index.html`: Contains the UI structure.
    - `style.css`: Contains CSS code for a nice interface.

2. **Run demo**:
    - Open the `index.html` file in your browser.

3. **Customize**:
    - Change the product list and orders directly in the HTML file.
    - To get interactive features like add/remove products or order processing, you will need to add JavaScript or backend programming (e.g. Node.js, PHP).
